Glassdesk

A comprehensive Claude Code framework — planning, multi-agent orchestration, SPARC methodology, and workflow automation.

Install

  1. Fork or clone the Glassdesk repository.
  2. Add it as a Claude Code marketplace: 
    claude plugin marketplace add <path-to-your-fork>
  3. Install the plugin: 
    claude plugin install glassdesk
  4. Verify: 
    claude plugin list

A short alias gd is also available: claude plugin install gd.

Capabilities

  • Planning — /plan, /plan:fast, /plan:hard, /plan:parallel
  • Implementation — /code, /code:auto, /code:no-test, /code:parallel
  • SPARC agents — 16 specialised commands under /sparc:*
  • Git integration — /git:status, /git:commit, /git:branch, /git:review
  • Fix workflows — 8 variants under /fix:* (CI, tests, types, UI…)
  • Utilities — /ask, /debug, /brainstorm, /scout
