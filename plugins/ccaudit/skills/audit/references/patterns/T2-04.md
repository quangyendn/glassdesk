# T2-04 — Permissions configured

## Why it matters
Running Claude Code without an explicit permissions allow-list or sandbox means it can write or execute anything it infers is needed. An allow-list forces review.

## Detection
Check `permissions.allow` length ≥ 3 OR `sandbox: true` in either settings file.

## Target
≥ 3 explicit allow entries OR sandbox enabled.
