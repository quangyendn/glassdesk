# T2-05 — CLI tools available

## Why it matters
`gh` CLI is required for most Claude Code git/PR workflows. Missing it forces manual steps that degrade AI-assisted development quality.

## Detection
`command -v gh` check. Informational check for `aws`, `gcloud`, `sentry-cli` (stack-specific).

## Target
`gh` present. Optionally: `aws`/`gcloud`/`sentry-cli` if used by the project.
