# T1-09 — Plugin auto-update redundancy

## Why expensive
~50 tokens per plugin × every session start = compounding overhead. Many plugins emit "loaded successfully" messages.

## Detection
Count `SessionStart` hook entries in user and project `settings.json`.

## Target
≤ 2 SessionStart hooks (branch context, env vars only).
