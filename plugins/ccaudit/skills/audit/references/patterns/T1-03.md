# T1-03 — Hook injection waste

## Why expensive
Each hook injects context (branch, files, instinct, memory) before Claude reads the user prompt.

## Detection
Count `UserPromptSubmit` entries in both user and project `settings.json`.

## Target
≤ 1 hook total (e.g. only git branch context).
