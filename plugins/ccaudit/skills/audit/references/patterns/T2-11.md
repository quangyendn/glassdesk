# T2-11 — Auto-memory awareness

## Why it matters
Claude Code auto-memory saves entries under `~/.claude/projects/<sanitized-cwd>/memory/`. If CLAUDE.md is already large (> 800 words) and auto-memory entries exist, the two sources may duplicate rules or conflict.

## Detection
Sanitize cwd with `pwd | sed 's|/|-|g'` (matches `~/.claude/projects/-Users-...` path convention). Count entries in that directory. Emit 1 if entries > 0 AND CLAUDE.md > 800 words.

Note: path sanitization convention may vary across Claude Code versions. If T2-11 silently emits 0, check your actual `~/.claude/projects/` directory manually.

## Target
Informational. If flagged, run `/memory` to review auto-saved entries and reconcile with CLAUDE.md.
