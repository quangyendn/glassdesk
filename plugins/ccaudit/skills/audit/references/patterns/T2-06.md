# T2-06 — Skill frontmatter integrity

## Why it matters
Claude Code discovers skills via `name` and `description` YAML frontmatter in SKILL.md. Missing fields mean the skill never loads.

## Detection
Parse SKILL.md files in both user and project skills directories. Check for `name` and `description` frontmatter keys.

## Target
100% of SKILL.md files have non-empty `name` and `description`.
