# T1-08 — Wrong-direction generation

## Why expensive
Output tokens are billed. Wasted output = direct cost. Letting a 400-line response finish even after 50 lines clearly drift.

## Detection
Self-observation. Run a 1-week habit log. Emits `manual` — human review required.

## Target
Use Cmd+. (Mac) / Ctrl+. within 5 seconds of detecting drift.
