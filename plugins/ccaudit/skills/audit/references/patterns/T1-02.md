# T1-02 — Conversation history re-reads

## Why expensive
Message N pays to re-tokenize messages 1..N-1. Cost grows linearly per turn → quadratic per session.

## Detection
Habit audit. No reliable log unless proxy is in place. Emits `manual` — human review required.

## Target
Hard cap 20 messages/session. Use `/compact` (not `/clear`) to retain summary.
