# T2-10 — Duplicate user vs project rules

## Why it matters
Lines that appear in both `~/.claude/CLAUDE.md` and `.claude/CLAUDE.md` are loaded twice every session. They waste tokens and create maintenance confusion about which copy is authoritative.

## Detection
Exact line match diff between user and project CLAUDE.md (ignoring blanks and comments), using `comm`.

## Target
0 duplicate lines. Move shared rules to user-level only; project-level holds project-specific overrides.
