# T1-06 — Just-in-case MCP tool definitions

## Why expensive
Each MCP ships its full tool schema (~600-1,200 tokens) on every request.

## Detection
Count `mcpServers` keys across both user and project `settings.json`.

## Target
3 always-on MCPs. Per-session enable for the rest.
