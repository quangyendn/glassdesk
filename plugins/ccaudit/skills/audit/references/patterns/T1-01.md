# T1-01 — CLAUDE.md bloat

## Why expensive
Loaded every turn, every session. Multiplied by ~200 turns/week.

## Detection
Scans every existing CLAUDE.md memory layer (see helper `_claude_md_files`): enterprise → user → project root → legacy `.claude/CLAUDE.md` → `CLAUDE.local.md`. Sums word counts across all layers found.

## Target
Combined < 1,200 words (~1,500 tokens).
