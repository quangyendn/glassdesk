# T1-05 — Skill loading on irrelevant tasks

## Why expensive
Each SKILL.md is 800-2,500 tokens. Conservative auto-load = "when in doubt, load".

## Detection
Count directories under `~/.claude/skills/`.

## Target
3-5 skills matching daily work.
