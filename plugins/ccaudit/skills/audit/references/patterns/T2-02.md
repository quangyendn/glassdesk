# T2-02 — @-imports scoped

## Why it matters
@-imports embed entire file contents into the context. Importing large docs (API references, full READMEs) balloons token usage on every turn.

## Detection
Scans every existing CLAUDE.md memory layer (see helper `_claude_md_files`). Parses @-prefixed lines, measures each imported file's word count (~1.3× for token estimate). Flags any single target > 1,500 tokens. Note: @-import recursion is not currently followed.

## Target
Each @-import target < 1,500 tokens. Total resolved context < 1,200 words.
