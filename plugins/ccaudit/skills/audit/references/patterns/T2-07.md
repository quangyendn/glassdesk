# T2-07 — Subagents present

## Why it matters
Anthropic recommends configuring subagent definitions for parallel task delegation. Absence is not a failure — it is informational: the user may not have adopted the pattern yet.

## Detection
Check for `.claude/agents` or `~/.claude/agents` directory existence.

## Target
Informational. Consider adding subagent definitions if parallel workflows are common.
